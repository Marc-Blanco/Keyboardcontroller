# Keyboardcontroller 
It´s a keyboard with the objective of being a leverless controller inspired by the arcade controllers specially designed for playing fgc games. All controlled with a raspberrypi PICO. It's main prupose its to serve as a controller for Street fighter, Tekken, etc... But it's sure that I will find more uses to it. It's Inspired by [Hitbox controllers](https://www.hitboxarcade.com/) and I do plan on making a levered version of it, it could be interpreted as a Keyboard extension also.
